Google’s new reCAPTCHA has a dark side

  • reCaptcha v3에 google cookie 가 판단기준에 사용되는 것 같고 google 계정 로그인 한 경우 lower risk score, Tor or VPN - higher score 나온다고. (higher – 사람, lower – 기계)
  • reCaptcha v3 의 경우 login 뿐만 아니라 전 페이지에 google js 넣기를 권고하고 있어 구글이 더 많은 개인정보를 수집할 수 있다는 우려
  • 수집되는 정보에는 hardware and software information 둘다 포함.
  • 기사가 링크가 논문에는 recaptcha v3가 mouse 움직임을 로봇 여부 판단 기준에 사용하는 것 같다고 함.

Network Obserability with Envoy

Upstream/Downstream 용어가 익숙치 않아 슬라이드의 내용 정리. * Listener: port, unix domain sockets * Downstream: a downstream host connects to envoy, send requests, receives responses. * Upstream: host sends requests from Envoy to other services and return responses * Cluster: group of logically similar upstream hosts that envoy connects to. jk - Configuration as code with TypeScript

typecheck, format check 쉽지 않은 yaml대신에 typescript 작성 후 yaml generate 하는 아이디어 vscode 등에서 자동완성도 지원할 수 있다고. json schema처럼 schema가 있어야 할것 같은데 안보이는게 의문.

// Alice is a developer.
const alice = {
  name: 'Alice',
  beverage: 'Club-Mate',
  monitors: 2,
  languages: [
    'python',
    'haskell',
    'c++',
    '68k assembly', // Alice is cool like that!
  ],
};

// Instruct to write the alice object as a YAML file.
export default [
  { value: alice, file: `developers/${alice.name.toLowerCase()}.yaml` },
];

Run this example with:

$ jk generate -v alice.js
wrote developers/alice.yaml

This results in the developers/alice.yaml file:

beverage: Club-Mate
languages:
- python
- haskell
- c++
- 68k assembly
monitors: 2
name: Alice

Why we’re ending support for MySQL in 12.1

  • We can’t support nested groups with MySQL in a performant way
    • With 구문.
  • We have to use hacks to increase limits on columns and this can lead to MySQL refusing to store data
  • MySQL can’t add TEXT type column without length specified
  • MySQL doesn’t support partial indexes

    ON events (measurement_id)
    USING btree
    (eventBody)
    WHERE is_active;
  • These limitations have already created a number of places where MySQL was already not supported (including with Geo)
